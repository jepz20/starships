### Prerequesite

To run this project locally you would need to have install:

1. node >= v12
2. yarn

### Run locally

install dependencies

```
yarn install
```

start the development server

```
yarn start
```

run tests

```
yarn test
```

run tests with coverage

```
yarn test -- --coverage
```
